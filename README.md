# Parallel Programming Labs

Лабораторні роботи з курсу "Паралельне програмування" (C# / .NET MAUI).

## Передумови

- .NET SDK 10.0+
- MAUI Windows workload: `dotnet workload install maui-windows`

## Запуск лабораторної

```powershell
dotnet run --project Lab1\Lab1.csproj
```

Або після `dotnet build Lab1\Lab1.csproj` запустити exe напряму:

```
.\Lab1\bin\Debug\net10.0-windows10.0.19041.0\win-x64\Lab1.exe
```

## Лабораторні

- **Lab1** — Робота з задачами та потоками. N задач рухають прямокутники у вікні; клік мишею всередині прямокутника завершує відповідну задачу. Кількість задач задається на старті та змінюється під час виконання.
- **Lab2** — Процеси. Запуск Excel із заданим файлом як параметром; неблокувальне очікування завершення процесу через подію `Process.Exited` із виведенням повідомлення "Файл *name* відредагований в Excel".
- **Lab3** — Синхронізація. Вікно ділиться на 4 квадранти; клік на 1-му генерує випадковий колір, три фонові потоки застосовують R/G/B-компоненти до решти квадрантів. Синхронізація через `lock` і `AutoResetEvent`.
- **Lab4** — Відображення файлу у пам'яті. Дві програми (`Lab4A` + `Lab4B`); коло у Lab4A рухається L→R, на правому краї через **memory-mapped file** + `Mutex` + `EventWaitHandle` передає `(Y, radius)` у Lab4B, де коло того ж розміру і висоти стартує свій рух L→R. `Lab4A` автоматично запускає `Lab4B.exe`.
